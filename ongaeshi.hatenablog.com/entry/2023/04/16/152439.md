を Windows からもやりたい。

PowerShell で標準入力を終了するにはCtrl+Z Enter なので・・

```
PS> ~/Documents/blog
PS> blogsync post --draft --title=blogsync ongaeshi.hatenablog.com
さてかきはじめるか...
(Ctrl+Z Enter)
```

これで OK。
