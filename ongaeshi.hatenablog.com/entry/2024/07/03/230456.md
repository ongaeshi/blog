Ruby と thor を使って書いた。[Runa](https://ongaeshi.hatenablog.com/entry/2023/07/23/113420) を使って blog コマンドとして登録してある。

[https://github.com/ongaeshi/blog_command:embed:cite]

# 使い方

```
# 新規記事作成
$ blog new タイトル

# VSCode で開く
$ blog edit

# 記事が完成したら git commit
$ git commit

# blogsync push
$ blog push

# blogsync pull
$ blog pull

# 開く
$ blog open
```
